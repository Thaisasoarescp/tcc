# 📊 Análise de Sentimentos em Repostagens sobre Mudanças Climáticas

## 📖 Sobre o Projeto
Este projeto de TCC tem como objetivo realizar a análise de sentimentos em repostagens coletadas por meio de técnicas de web scraping. A pesquisa está inserida no contexto de Ciência de Dados e Processamento de Linguagem Natural (PLN), buscando compreender a percepção pública sobre mudanças climáticas e desastres naturais. É um projeto realizado em grupo para a Universidade Virtual do Estado de São Paulo.

## 🎯 Objetivos
- Coletar dados de repostagens em plataformas digitais utilizando web scraping
- Aplicar técnicas de processamento de linguagem natural (PLN)
- Classificar os sentimentos (positivo, negativo, neutro)
- Identificar padrões e tendências nas opiniões públicas

## 🛠️ Tecnologias Utilizadas
- Python
- Pandas
- NumPy
- Scikit-learn
- BeautifulSoup / Selenium
- Matplotlib / Seaborn
